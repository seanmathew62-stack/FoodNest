Reward System

If the hosteller gives the order a 5-star review, the total NestCoins of that order will be shared equally between:

  • The Hosteller
  • The Chef
  • The Delivery Agent

Example: If an order is worth 60 NC and receives 5 stars:

  • Chef gets 20 NC
  • Delivery Agent gets 20 NC
  • Hosteller gets 20 NC back

This makes the system fun, rewarding, and beneficial for everyone involved.
